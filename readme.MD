# GSW TEST FRONTEND

### Informações

Servidor está rodando na porta 8085

### Tecnologias

Todas tecnologias que utilizada são open source:

* AngularJS
* Twitter Bootstrap
* Less
* Bower
* Npm
* Gulp

### Instalação

Instalar as dependencias e dependencias de desenvolvimento e rodar o servidor

```sh
$ cd gsw-test-frontend
$ npm install
$ bower install
$ gulp
```
